The Cable Network

Cabernet allows you to be in control of the IPTV streams. Elegant yet loaded with functionality, Cabernet is the future of streaming services.

Learn More Download

Connect streams to your favorite media server. Cabernet is a modular-based appliance/platform that cleans, organizes and repackages IPTV streams to be compatible and digested by media clients.

Clients

M3U and HDHomerun interfaces integrate with media servers. Tested clients include:

  • TVheadend
  • Emby
  • Jellyfin
  • Plex
  • VLC

EPG Management

  • Merges xmltv data
  • Automatic XMLTV/EPG updates

Channel Management

  • Merges M3U data
  • Channel filtering
  • Renaming Channel number and name
  • Adding Channel categories/groups
  • Updating Channel icons/logos
  • Automatic channel updates

Streaming

  • Support for M3U8 using FFMPEG or Internal
  • Support for M3U direct proxy with media server
  • ATSC correcting software for corrupted streams
  • Adjustable and Extensible number of tuners using multiple instances
  • Stream filtering

General

  • Backup and restore
  • Simple python deployment (3.7 or greater)
  • Limited use of external libraries
