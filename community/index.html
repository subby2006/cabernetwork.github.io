Join the Cabernet community

Cabernet is an open source project that anyone in the community can use, improve, and enjoy. We'd love you to join us! Here's a few ways to find out what's happening and get involved.

Learn and Connect

Using or want to use Cabernet? Find out more here:

  • Documentation: Find out how to use Cabernet
  • Github Issues: Practical questions and curated answers. Report bugs here!

Develop and Contribute

If you want to get more involved by contributing to Cabernet, join us here:

  • GitHub: Development takes place here!
  • Discord: Chat with other project developers and maintainers
  • Forum (archived): Discuss development issues around the project

You can find out how to contribute to these docs in our Contribution Guidelines.
